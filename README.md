# Better Hide Hotbar Mod

Простой и легковесный клиентский модификатор для Minecraft (Fabric), предназначенный для точечного скрытия элементов
игрового интерфейса (HUD).

Мод позволяет полностью убрать нижнюю панель предметов, статус-бары и полоску опыта, оставляя при этом важные элементы
вроде прицела, чата и руки персонажа нетронутыми. Отлично подходит для создания красивых скриншотов, записи
кинематографичных видео (машиним) или для усложненного хардкорного геймплея.

## Выбор версии исходного кода (Старая версия)

> [!WARNING]
> Используйте main ветку, ветки 1.21.7 и 1.21.11 останутся без поддержки

Архитектура игрового интерфейса Майнкрафта сильно менялась в версиях 1.21+. Выберите ветку репозитория, соответствующую
вашей версии игры, чтобы посмотреть нужный исходный код миксинов:

* **[Исходный код для Minecraft 1.21.7](https://github.com/themaster1970sf/HideHotbar/tree/1.21.7)** — использует
  базовое скрытие через `renderMainHud` и `renderStatusBars`.
* **[Исходный код для Minecraft 1.21.11](https://github.com/themaster1970sf/HideHotbar/tree/1.21.11)** — адаптирован под
  новые маппинги с дополнительным инжектом в метод `renderExperienceBar`.

## Функционал

При нажатии на горячую клавишу **F7** (можно переназначить в настройках управления) на экране скрываются/появляются
следующие элементы:

* **Хотбар** (горячая панель ячеек предметов).
* **Статус-бары** (здоровье, голод, броня, пузырьки воздуха, щиты).
* **Опыт** (как зеленая полоска прогресса, так и цифра текущего уровня).

Присутствует кастомизация, то есть можно изменить что останется на экране, а что пропадет

**Что ОСТАЕТСЯ видимым:**

* Перекрестие прицела (Crosshair).
* Игровой чат и список игроков на Tab.
* Рука персонажа (и предметы в ней).
* Эффекты зелий в правом верхнем углу экрана.

При желании можно настроить появление хотбара при определенных условиях, например:

* При получении урона
* При смене слота
* При перезапуске игры
  И другие (итд - топ)

## Как собрать проект локально

Если вы хотите собрать `.jar` файл мода из исходного кода:

1. Склонируйте репозиторий и переключитесь на нужную ветку:

   ```bash
   git clone https://github.com/themaster1970sf/HideHotbar.git
   cd HideHotbar
   # если не видите common/ директории или использовали старые версии, то используйте
   git checkout main  
   ```


2. Скомпилируйте мод с помощью Gradle:

```bash
./gradlew build # Для сборки всех версий
```

или

```bash
# 1_21_11 можно заменить на 1_21_7
# Посмотрите доступные версии для актуальной информации
./gradlew :mc1_21_11:build 
```

[Доступные версии](#доступные-версии)

3. Готовый `.jar` файл появится в папке `mc1_21_11/build/libs/`.
