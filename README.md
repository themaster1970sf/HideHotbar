# Better Hide Hotbar Mod

Простой и легковесный клиентский модификатор для Minecraft (Fabric), предназначенный для точечного скрытия элементов игрового интерфейса (HUD). 

Мод позволяет полностью убрать нижнюю панель предметов, статус-бары и полоску опыта, оставляя при этом важные элементы вроде прицела, чата и руки персонажа нетронутыми. Отлично подходит для создания красивых скриншотов, записи кинематографичных видео (машиним) или для усложненного хардкорного геймплея.

---

##  Выбор версии исходного кода

Архитектура игрового интерфейса Майнкрафта сильно менялась в версиях 1.21+. Выберите ветку репозитория, соответствующую вашей версии игры, чтобы посмотреть нужный исходный код миксинов:

*  **[Исходный код для Minecraft 1.21.7](https://github.com/themaster1970sf/HideHotbar/tree/1.21.7)** — использует базовое скрытие через `renderMainHud` и `renderStatusBars`.
*  **[Исходный код для Minecraft 1.21.11](https://github.com/themaster1970sf/HideHotbar/tree/1.21.11)** — адаптирован под новые маппинги с дополнительным инжектом в метод `renderExperienceBar`.

##  Функционал

При нажатии на горячую клавишу **F7** (можно переназначить в настройках управления) на экране скрываются/появляются следующие элементы:
*  **Хотбар** (горячая панель ячеек предметов).
*  **Статус-бары** (здоровье, голод, броня, пузырьки воздуха, щиты).
*  **Опыт** (как зеленая полоска прогресса, так и цифра текущего уровня).

**Что ОСТАЕТСЯ видимым:**
*  Перекрестие прицела (Crosshair).
*  Игровой чат и список игроков на Tab.
*  Рука персонажа (и предметы в ней).
*  Эффекты зелий в правом верхнем углу экрана.

Если включена функция **unhide-on-damage**, то при уроне хотбар становится видимым (Отключено изначально)

##  Как собрать проект локально

Если вы хотите собрать `.jar` файл мода из исходного кода:

1. Склонируйте репозиторий и переключитесь на нужную ветку:
   
   ```bash
   git clone https://github.com/themaster1970sf/HideHotbar.git
   cd HideHotbar
   git checkout 1.21.11  # или 1.21.7 в зависимости от версии
   ```


2. Скомпилируйте мод с помощью Gradle:
```bash
./gradlew build
```


3. Готовый `.jar` файл появится в папке `build/libs/`.
